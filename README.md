# 넷플릭스 클론 사이트
동영상 재생을 제외한 인터페이스 만을 구현한 넷플릭스 클론 사이트 입니다.

# 기능
### 1. 영화, TV show 모아보기
영화와 TV쇼를 카테고리에 따라서 모아볼 수 있습니다.
또한 각 영화나 Tv show 에 대한 자세한 정보도 알아볼 수 있습니다.
         
### 2. 검색
키워드를 통한 검색을 통해 원하는 영화나 티비쇼를 찾아볼 수 있습니다.

# 사용기술 스택
Typescript, React, styled-componenet, framer-motion
